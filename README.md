# Digital-Time

---

## DigitalTime

A modern and interactive clock with real-time visualization, combining an analog clock, digital clock, and a dynamic 3D Earth background.

---

## Features

- Real-time digital clock  
- Synchronized analog clock  
- 3D Earth rendering (WebGPU)  
- Timezone selection  
- Theme toggle (light/dark)  
- Integrated search bar (Google)  
- Timer with:
  - Start  
  - Pause  
  - Reset  
- Fully responsive layout  
- Optimized for performance and smoothness  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- WebGPU API  
- JavaScript modularization
